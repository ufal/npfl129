# Support Vector Machines slides

This presentation is written using [Typst](https://typst.app/) and [Polylux](https://sitandr.github.io/polylux/book/polylux.html) with the [ÚFAL template](https://github.com/kasnerz/typst-slides-ufal). 

## Quickstart

1. **Install Typst**:
   ```bash
   # Using cargo
   cargo install --git https://github.com/typst/typst
   
   # Or download from GitHub releases
   ```
2. **Install the fonts (if you don't have them yet)**
   - See `fonts/Fira Sans` and `fonts/Consolas`.
3. **Export the slides**
   ```bash
   typst compile svm.typ svm.pdf
   ```
