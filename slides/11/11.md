# Singular Value Decomposition

Every (even a rectangular) matrix $â†’X$ of dimenion $m \times n$ and rank $r$ can be factorized as

$$ â†’X = â†’U â†’\Sigma â†’V^T $$

* $â†’U$ is a $m \times m$ orthonormal matrix

* $â†’\Sigma$ is a $m \times n$ diagonal matrix with non-negative values, so-called singular values, chosen to be in the decreasing order

* $â†’V$ is a $n \times n$ orthonormal matrix

$$
â†’X â†’V = â†’U â†’\Sigma

\quad\quad\Rightarrow\quad\quad

â†’X â†’v_k = \sigma_k â†’u_k \quad \forall k = 1, \ldots, r
$$

$$ â†’X \left[\begin{array}{c} \vdots \\ â†’v_1 \cdots â†’v_r \cdots â†’v_n \\ \vdots \end{array}\right] = \left[\begin{array}{c} \vdots \\ â†’u_1 \cdots â†’u_r \cdots â†’u_m \\ \vdots \end{array}\right] \left[\begin{array}{ccc|c}\sigma_1 \\ & \ddots & & â†’0 \\ & & \sigma_r \\ \hline & â†’0 & & 0 \end{array}\right]$$

# SVD: Proof

Assuming SVD exists, we can write $â†’U$ and $â†’V$ as eigenvector decomposition of row and column similarities:

$$
â†’X â†’X^T = (â†’U â†’\Sigma â†’V^T) (â†’U â†’\Sigma â†’V^T)^T =
â†’U â†’\Sigma (â†’V^T â†’V) â†’\Sigma â†’U^T = â†’U â†’\Sigma â†’\Sigma^T â†’U^T
$$

$$
â†’X^T â†’X = (â†’U â†’\Sigma â†’V^T)^T (â†’U â†’\Sigma â†’V^T) =
â†’V â†’\Sigma (â†’U^T â†’U) â†’\Sigma â†’V^T = â†’V â†’\Sigma^T â†’\Sigma â†’V^T
$$

~~~
Let's take $â†’V = [â†’v_1, \ldots, â†’v_r]^T$ orthonormal eigenvectors of $â†’X^T â†’X$ and set $\sigma_k = \sqrt{\lambda_k}$, then, $â†’X^T â†’X â†’v_k = \sigma_k^2 â†’v_k$. ($\lambda_k \geq 0$ because $â†’X â†’X^T$ is positive definite.)

~~~
The decomposition says that $â†’X â†’v_k = \sigma_k â†’u_k \Rightarrow â†’u_k = \frac{â†’X â†’v_k}{\sigma_k}$. To make it work, we need to show that $â†’u_k$ is indeed an eigenvector of $â†’X â†’X^T$ with the same eigenvalue $\lambda_k$.

~~~
$$
â†’X â†’X^T â†’u_k =

â†’X â†’X^T \underbrace{\left( \frac{â†’X â†’v_k}{\sigma_k} \right)}_{\text{def. of $â†’u_k$}} =

â†’X \underbrace{\left( \frac{â†’X^T â†’X â†’v_k}{\sigma_k} \right)}_{â†’v_k \text{ is eigenvector of } â†’X^T â†’X} =

â†’X \frac{\sigma_k^2 â†’v_k}{\sigma_k} =

\sigma_k^2 \underbrace{\left( \frac{â†’X â†’v_k}{\sigma_k} \right)}_{\text{def. of $â†’u_k$}} =

\sigma_k^2 â†’u_k
$$

# Interpretation of SVD

* Vectors of $â†’U$ the components rows consists of, vectors of $â†’V$ are the same for the colums

~~~
* It defines low-rank decomposition of $â†’X$ (with rank $r$) as a sum of rank 1 matrices of dimension $m \times n$

$$ â†’X = \sigma_1 â†’u_1 â†’v_1^T + \sigma_2 â†’u_2 â†’u_2^T + \ldots + \sigma_r â†’u_r v_r^T $$

$$\tilde{â†’X} = \sum_{i=1}^k \sigma_i u_i v_i^T$$

# Eckart-Young Theorem

$â†’X \in \mathbb{R}^{n \times m}$ and $â†’X_k = \sigma_1 â†’u_1 â†’v_1^T + \ldots + \sigma_k â†’u_k â†’v_k^T$ its approxiamation using SVD. For each $â†’B$

$$
|| â†’X - â†’X_k ||_F \leq || â†’A - â†’B ||_F.
$$

## Sketch of the proof

* $|| â†’X ||_F = \sqrt{\sum_{i}^n\sum_{j}^m x_{ij}^2} = \sqrt{\operatorname{trace}(â†’X^T â†’X)}$ (trace is the sum of the diagonal)

* Multiplying by orthonormal matrix does not change the norm

$$
||â†’U â†’A ||_F^2 =
\operatorname{trace}((â†’Uâ†’A)^T â†’Uâ†’A) =
\operatorname{trace}(â†’A^T \underbrace{â†’U^T â†’U}_{â†’I} â†’A) =
\operatorname{trace}(â†’A^T â†’A) = ||â†’A||_F^2
$$

* The Frobenius norm is the $L^2$ norm of the diagonal of $â†’\Sigma$

* The best strategy to keep the most of the norm is removing the smallest values

# From SVD to Principal Component Analysis

So far, SVD had geometric interpretation, let's add statistical interpretation.

When we apply SVD on mean-centered data $â†’X - \bar{â†’x}$, singular value get new interpretation: **components explaining variability in the data**.

$$
||â†’X - \bar{â†’x}||_F^2 =
\operatorname{trace}\underbrace{\left((â†’X - \bar{â†’x})^T(â†’X - \bar{â†’x})\right)}_{N\operatorname{Cov}(â†’X)}  =
N \sum_i^D \operatorname{Var}( â†’X_{i,:} )
$$

Approximating the matrix in terms of Frobenius norm means keeping the most variance from the data. Components ordered by how much variablity in the data they capture.

~~~
<br />

Let $â†’S = \frac{1}{N}(â†’X - \bar{â†’x})^T(â†’X - \bar{â†’x})$, then PCA of $â†’X$ are the eigenvectors of $â†’S$, i.e., the $â†’V$ matrix of the SVD decomposition of $â†’X - \bar{â†’x}$.

Note the $\frac{1}{N}$ term scales down the eigenvalues compared compared to SVD, but keeps the eigenvectors unchanged.

# PCA Applications â€“ Whitening aka Sphering

The PCA formula allows us to perform **whitening** aka **sphering**, which is
a linear transformation of the given data so that the resulting dataset
has zero mean and an identity covariance matrix.

~~~

$$â†’z_i â‰ â‡‰\Sigma^{-1} â‡‰V^T (â†’x_i - â†’xÌ„).$$

~~~
Then, the mean of $â†’z_i$ is zero and the covariance is given by
$$\begin{aligned}

  &= \frac{1}{N} âˆ‘_i â‡‰\Sigma^{-1} â‡‰V^T (â†’x_i - â†’xÌ„) (â†’x_i - â†’xÌ„)^T â‡‰V â‡‰\Sigma^{-1} \\

# Computing PCA â€” The Power Iteration Algorithm

If we want only the first (or several first) principal components, we might use
the **power iteration algorithm**.

~~~
The power iteration algorithm can be used to find a **dominant** eigenvalue (an

compute PageRank). It works as follows:

<div class="algorithm">

**Input**: Real symmetric matrix $â‡‰A$ with a dominant eigenvalue.<br>
**Output**: The dominant eigenvalue $Î»$ and the corresponding eigenvector $â†’v$, with
probability close to 1.

- Initialize $v$ randomly (for example each component from $U[-1,1]$).
~~~
- Repeat until convergence (or for a fixed number of iterations):
  - $â†’v â† â‡‰Aâ†’v$
  - $Î» â† \|â†’v\|$
  - $â†’v â† â†’v / Î»$
</div>

~~~
If the algorithm converges, then $â†’v = â‡‰Aâ†’v / Î»$, so $â†’v$ is an eigenvector with
eigenvalue $Î»$.

In order to analyze the convergence, let $(Î»_1, Î»_2, Î»_3, â€¦)$ be the eigenvalues
of $â‡‰A$, in the descending order of absolute values, so $|Î»_1| > |Î»_2| â‰¥ |Î»_3| â‰¥ â€¦$,

~~~

$$â‡‰Aâ†’v = \left(Î»_ 1 a_ 1, Î»_ 2 a_ 2, Î»_ 3 a_ 3, â€¦\right) $$

$$â‡‰A^kâ†’v = \left(Î»_1^k a_1, Î»_2^k a_2, Î»_3^k a_3, â€¦\right) = \lambda_1^k \left(a_1, \frac{\lambda_2^k}{\lambda_1^k}, \frac{\lambda_3^k}{\lambda_1^k}, \ldots\right)$$

After we get the largest eigenvalue $Î»_1$ and its eigenvector $â†’v_1$, we can modify the

# K-Means Clustering

Let $â†’x_1, â†’x_2, â€¦, â†’x_N$ be a collection of $N$ input examples, each being
a $D$-dimensional vector $â†’x_i âˆˆ â„^D$. Let $K$, the number of target clusters,
be given.

~~~
Let $z_{i, k} âˆˆ \{0, 1\}$ be binary indicator variables describing whether an input
example $â†’x_i$ is assigned to cluster $k$, and let each cluster be specified by
a point $â†’Î¼_1, â€¦, â†’Î¼_K$, usually called the cluster **center**.

~~~
Our objective function $J$, which we aim to minimize, is
$$J = âˆ‘_{i=1}^N âˆ‘_{k=1}^K z_{i, k} \|â†’x_i - â†’Î¼_k\|^2.$$

---
# K-Means Clustering

<div class="algorithm">

**Input**: Input points $â†’x_1, â€¦, â†’x_N$, number of clusters $K$.

- Initialize $â†’Î¼_1, â€¦, â†’Î¼_K$ as $K$ random input points.

~~~
- Repeat until convergence (or until patience runs out):
~~~
  - Compute the best possible $z_{i, k}$. It is easy to see that the smallest $J$
    is achieved by
    $$z_{i,k} = \begin{cases} 1 & \textrm{~~if~}k = \argmin\nolimits_j \|â†’x_i - â†’Î¼_j\|^2, \\
                              0 & \textrm{~~otherwise}.\end{cases}$$

~~~
  - Compute the best possible $â†’Î¼_k = \argmin\nolimits_{â†’Î¼} âˆ‘_i z_{i,k} \|â†’x_i-â†’Î¼\|^2$.
~~~
   By computing a derivative with respect to $â†’Î¼$, we get
   $$â†’Î¼_k = \frac{âˆ‘_i z_{i,k} â†’x_i}{âˆ‘_i z_{i,k}}.$$
</div>

It is easy to see that:
- updating the cluster assignment $z_{i, k}$ decreases the loss $J$ or keeps it the same;
~~~
- updating the cluster centers again decreases the loss $J$ or keeps it the
  same.

~~~
K-Means clustering therefore converges to a local optimum. However, it
is quite sensitive to the starting initialization:
~~~
- It is common practice to run K-Means algorithm multiple times with different
  initialization and use the result with the lowest $J$ (scikit-learn uses
  `n_init=10` by default).
~~~
- Instead of using random initialization, `k-means++` initialization scheme might
  be used, where the first cluster center is chosen randomly and others are
  chosen proportionally to the square of their distance to the nearest cluster
  center. It can be proven that with this initialization, the solution
  has $ğ“(\log K)$ approximation ratio in expectation.
